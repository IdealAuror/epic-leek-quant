## 四、关键教训

1. **A 股动量窗口比美股短**：40 天有效（61-21 momentum），非传统 12-1 月
2. **回撤约束多余**：无风控版策略环境回撤 34.93% 已达标，回撤约束降低 Sharpe
3. **IC 局限性**：单因子截面排序无法衡量组合分散化价值（F5/F6 均验证）
4. **辅助因子必须低相关**：F4-ROE 与 EP 高相关→负增量；F5-LowVol 选股池不同→正增量
5. **跨样本验证不可省**：F1 仅在 AllA 有效→壳价值污染；F6 三样本通过→非 size proxy

## 五、策略演化逻辑

### 为什么不是传统烟蒂

原始格雷厄姆-施洛斯烟蒂策略的收益来源：

```
年化 ≈ (卖出PB − 买入PB) × 换手率 / 买入PB + ROE
     ≈ (1.0 − 0.6) × 25% / 0.6 + 5% ≈ 21.7%
     ≈ 75% PB 估值修复 + 25% 内生增长
```

前提是有大量 **PB < 0.6** 的烟蒂股。但 A 股流动性极度充裕，这类标的几乎不存在——钱太多了，任何低估值都会被迅速填平。

因此策略被迫从**绝对低估**转向**相对低估+趋势确认**：

| 维度 | 传统烟蒂 | 本策略 |
|------|---------|--------|
| 价值信号 | PB < 0.6（绝对阈值） | EP Z-score 排名（相对便宜） |
| 回归周期 | 4年（持有等待） | 2.3季度（季度轮动） |
| 安全边际 | 资产折价本身 | 低波动过滤 + 50只分散 |
| 趋势确认 | 无（纯逆向） | F6 动量（避开价值陷阱） |
| 换手逻辑 | 等待回归后卖出 | 每季度重新排名，持续轮动 |

### 收益拆解

回测年化约 **19%**（2014.01–2026.06，累计 776%）：

| 来源 | 贡献 | 机制 |
|------|------|------|
| **F2 · EP 价值轮动** | ~6–7% | 每季度买入相对便宜的股票，短期均值回归。不等"回到合理估值"，只要排名变化就换 |
| **F6 · 趋势动量** | ~3–4% | 40天趋势信号。避开持续阴跌的价值陷阱（如 2019-2020 抱团行情中的伪便宜股） |
| **F5 · 低波动溢价** | ~1–2% | 规避高波动"彩票股"，降低组合回撤 → 复利效应更强 |
| **分散再平衡** | ~0.5–1% | 50只等权 + 季度调仓 → 自然低买高卖 |
| **市场 Beta** | ~5.4% | β=0.777，防御属性，牛市跟涨、熊市抗跌 |

> 关键差异：传统烟蒂 75% 收益靠**一次性估值修复**（等 4 年），本策略把"修复"拆成了**每季度的小幅轮动**（不依赖极端低估，依赖持续相对排名）。

## 七、待探索参数（防过拟合红线）

以下参数有进一步优化的空间，但三样本验证已 PASS，任何改动必须有严格跨期/跨样本验证：

| 参数 | 当前值 | 可扫描范围 | 风险 |
|------|--------|-----------|------|
| **动量窗口** | 61-21（40天） | 80-20, 120-40, 150-50 等 | P5 已扫 4 个方向×2 极性，40d 全样本最优。进一步微调空间有限 |
| **因子权重** | 等权 1/3 | ICIR 加权 / Risk Parity | 最大过拟合风险点。等权是最稳健基准，ICIR 依赖样本内估计 |
| **持仓数** | 50 只 | 30 / 40 / 60 / 80 | 40-60 差异不大，<30 分散不足，>80 流动性小票稀释收益 |
| **调仓频率** | 季度(5/9/11) | 双月 / 半年 | 半年太慢跟不上轮动，双月手续费↑换手↑ |
| **波动率窗口** | 60 日 | 30 / 90 / 120 日 | 30 日太短噪音大，>90 日滞后 |
| **回撤保护阈值** | 20%→85% | 15%→90% / 25%→80% | 已放宽到省电模式，进一步放宽≈去掉风控 |
| **流动性门槛** | 1000万/日 | 2000万 / 5000万 | 提高门槛换流动性安全，可能牺牲小票 alpha |

### 安全扫描策略

```
1. 固定训练集 2014-2020，验证集 2021-2026（天然跨期）
2. 每个参数独立扫描，不改多参数组合（防维度灾难）
3. 只接受三样本（AllA / CSI300 / CSI500）同时改善的参数
4. 改善幅度需 > 2% Sharpe 或 > 3pp 年化才考虑切换
```

> **当前建议**：等权三因子 50 只已是经过 5 个 Phase 验证的最稳健配置。在无新因子加入的前提下，参数微调边际收益有限，过拟合风险大于潜在收益。
