## 四、关键洞察

### 洞察 1：FM 检验 ≠ 组合价值

F5 LowVol 在 FM 检验中不显著（单变量 t=0.95，多变量 t=0.90），但作为组合辅助因子价值极高：
- F2 单因子 527% → F2+F5 组合 1237%（收益翻倍）
- F2 单因子回撤 21% → F2+F5 组合回撤 18%（回撤降低）
- F2 单因子 Sharpe 0.51 → F2+F5 组合 Sharpe 0.77（提升 50%）

**结论**：FM 检验独立 alpha，但风险调节器的价值在于分散化，不在于独立 alpha。"FM 通过才能入组合"是教条。

### 洞察 2：Phase 1 IC 通过可能是 size 效应残留

F3 Dividend（Phase 1 IC t=2.74）和 F5 LowVol（Phase 1 IC t=3.17）在 Phase 2 控制市值后都不显著。这说明 Phase 1 的市值中性化可能不够彻底，或 IC 分析对 size 效应的敏感度低于 FM 回归。

**对 Phase 3 的启示**：Phase 3 Walk-Forward CV 应特别关注 size 因子的时变暴露，确认 alpha 不是 size proxy。

### 洞察 3：F5 风险调节器价值远超预期

F5 把 F2 收益翻倍+回撤降低，这是 Phase 1 未充分发现的价值。Phase 1 只看到"组合降回撤 4pp"，Phase 2 单因子对比才暴露出"收益翻倍"的惊人效果。

**对 Phase 3 的启示**：Phase 3 模块 4（目标波动率仓位管理）应基于 F5 信号做仓位调节，把 F5 从选股层延伸到仓位层，可能进一步降回撤。
