## 四、关键发现与解读

### 发现 1：F2+F5 Baseline 稳健性得到强验证

M1/M2/M3 三模块全通过，证明：
- **不过拟合**：样本外 Sharpe 0.563，外/内比率 0.70
- **跨断点稳健**：4 个 A 股重大制度断点后表现方向不反转
- **alpha 真实**：DSR 0.9679，F2 alpha 在 6 次多重检验后仍显著

这是 Phase 3 的核心价值——F2+F5 Baseline 不是 data-mining 产物，是真正稳健的策略。

### 发现 2：策略能自我修复

2017 MSCI 纳入后 Sharpe 暴跌（2.15→0.07），但 2019 注册制后恢复（0.07→1.02）。这说明策略对市场结构变化敏感，但因子逻辑（EP + LowVol）在新的市场结构下能重新生效。

**对 Phase 4 的启示**：实盘需监控策略表现，若 Sharpe 持续低于 0.5 超过 1 年，需重新评估因子有效性。

### 发现 3：策略近期表现恶化

第 7 轮 Walk-Forward（2025-2026）样本外 Sharpe -0.316，是唯一负值。可能原因：
- 2025 年 A 股市场结构变化（需进一步调查）
- F2 EP 因子在低利率环境失效
- 样本量小（1 年仅 3 个调仓日），统计噪声大

**对 Phase 4 的启示**：实盘启动后需密切监控 2025-2026 表现，若持续恶化需暂停加仓。

### 发现 4：目标波动率方法局限

目标波动率仓位管理降波动有效（32% → 14%），但降回撤无效（18.88% → 18.88%）。原因：
- 回撤是路径依赖的，目标波动率只控制单期波动
- 若策略连续亏损，仓位管理不会避免回撤（除非仓位降到 0）

**对 Phase 4 的启示**：需换回撤控制方法，如：
- 最大回撤约束（DD limit）
- 动态止损（trailing stop）
- CVaR 优化

### 发现 5：V1 = V2 的实现问题

F5 overlay 阈值 `vol_z < -1` 过严，从未触发。这暴露了 F5 信号叠加的实现问题：
- F5_vol 经市值中性化+winsorize 后方差变小
- z-score 阈值 -1 对应 16% 分位，但 37 个点中可能没有极端值

**对 Phase 4 的启示**：F5 overlay 阈值需调整到 -0.5（31% 分位）或用百分位数替代 z-score。

## 五、决策

### 5.1 Phase 3 终判

**3/4 模块通过**，按 spec 失败条件：
- "≤2 模块通过 → Phase 3 失败" → ❌ 不成立（3/4 通过）
- "3 模块通过（模块 4 未通过）→ 策略稳健但回撤问题未解决，进 Phase 4 但限制初始规模" → ✅ 成立

**终判**：Phase 3 部分通过，进 Phase 4 实盘校准，限制初始规模。

### 5.2 策略决策

- **当前最优策略仍为 F2+F5 等权 z-score**（P1-F2F5-EP-LowVol-2026Q2-v1）
- Phase 3 验证了 Baseline 的稳健性，**不替换 Baseline**
- 目标波动率仓位管理未通过，**不采用 TV 仓位管理**
- Phase 4 用 Baseline + 限制初始规模（5000 万而非 1 亿）

### 5.3 Phase 4 启动条件

1. **策略**：F2+F5 等权 z-score Baseline（无仓位管理）
2. **初始规模**：5000 万（限制规模，降低回撤绝对值）
3. **仓位管理**：Phase 4 重新设计（最大回撤约束 / 动态止损 / CVaR）
4. **监控重点**：2025-2026 策略表现，若 Sharpe 持续 < 0.5 超过 1 年需暂停
5. **容量评估**：5000 万资金下的冲击成本（持仓 50 只，单只约 100 万）
