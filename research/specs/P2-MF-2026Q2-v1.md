## 一、假设

```yaml
hypothesis: |
  Phase 1 通过的三个因子（F2-EP / F3-Dividend / F5-LowVol）中，存在一组经
  Fama-MacBeth 检验具备独立截面定价能力的子集，其 Risk Parity 合成组合相比
  当前等权 z-score Baseline（P1-F2F5-EP-LowVol-2026Q2-v1）：
    - Sharpe 不下降
    - 最大回撤不上升
    - 残差 alpha 显著 > 0（不被 size/value/momentum/low-vol 已知风险因子完全解释）

economic_intuition: |
  Phase 1 三因子经济定位：
    - F2-EP：估值因子（便宜股），收益主力，但成长股牛市回撤大
    - F3-Dividend：现金流因子（分红股），理论上下行保护，但 A 股断点后失效
    - F5-LowVol：风险管理因子（稳健股），与 EP 低相关，组合中作风险调节器

  Phase 2 三步筛选逻辑：
    (1) **相关性矩阵**：若 |Spearman(f_i, f_j)| > 0.6 则视为重复暴露，去重或合并。
        预期 F2 与 F3 中等相关（都是价值因子），F2/F3 与 F5 低相关。
    (2) **Fama-MacBeth**：每月截面回归 fwd_return ~ F2 + F3 + F5，看各因子
        系数 t 值。**重点验证 F3 是否有独立定价能力**（F3 全样本 t=1.49 较弱，
        断点后 t=0.41，可能被 F2 吸收）。FM 用于"筛选"而非"组合构建"。
    (3) **Risk Parity 合成**：基于 FM 通过的因子，用等风险贡献替代等权 z-score。
        风险低的因子权重大，避免 F2 单一主导风险。

  **回应 PROJECT-PLAN Phase 2 重点排除项**：
    - 国企=low-beta/size 代理：本项目 F4 = ROE（非国企），且已放弃；本项目未单独
      验证国企因子，Phase 2 不涉及。
    - 回购=无效：本项目未启动回购因子方向，Phase 2 不涉及。

failure_condition: |
  明确失败条件：
  - 相关性矩阵出现 |corr| > 0.8 的强重复暴露 → 去重后剩 <2 个因子，组合无意义
  - Fama-MacBeth 中所有因子系数 |t| < 2 → 无独立定价能力，Phase 1 是 size proxy 共线
  - Risk Parity 合成相比 Baseline：Sharpe 下降 OR 回撤上升 → RP 不优于等权
  - 残差 alpha ≤ 0 → 合成组合收益被已知风险因子完全解释，无真实 alpha
  - 行业中性化后超额收益显著下降 → 选股能力来自行业配置，非行业内选股
```

## 五、Phase 2 三步分析方法

```yaml
analysis:
  step1_correlation_matrix:
    description: |
      因子两两 Spearman Rank 相关性矩阵。
      每个调仓日横截面上计算 rank(F_i) 与 rank(F_j) 的 Spearman 相关，
      再对全时段取均值与标准差，观察稳定性。
    threshold: |
      |corr_mean| > 0.6 → 重复暴露，需去重或合并
      |corr_mean| > 0.8 → 强重复，必须二选一
    outputs:
      - "3x3 相关性矩阵（均值）"
      - "3x3 相关性矩阵（标准差，看稳定性）"
      - "相关性时间序列（看是否系统性漂移）"

  step2_fama_macbeth:
    description: |
      Fama-MacBeth 两阶段回归：
      第一阶段（截面回归）：每月 t，
        fwd_return_i = γ0_t + γ1_t*F2_i + γ2_t*F3_i + γ3_t*F5_i + ε_i
        其中 F_i 已做横截面 Z-score 标准化（系数可比）
      第二阶段（时序平均）：
        γ_k = mean(γ_k_t), t-stat = γ_k_mean / (std(γ_k_t) / sqrt(T))
        Newey-West 调整 t（滞后 4 期）
    thresholds:
      - "保留因子：|FM t| ≥ 2（独立截面定价能力）"
      - "剔除因子：|FM t| < 2（被其他因子吸收，无独立贡献）"
    expected_check: |
      F3 是否被 F2 吸收（F3 全样本 t=1.49 弱，断点后 t=0.41），
      若 FM 中 F3 系数 |t| < 2 则剔除 F3，Phase 2 退化为 F2+F5 双因子 RP。
    variants:
      - "全样本 FM（2014-2026）"
      - "断点前 FM（2014-2019.06）"
      - "断点后 FM（2019.06-2026）"
      - "断点前后系数稳定性"
    outputs:
      - "FM 系数表（F2/F3/F5 各列：均值/NW-t/正负比例）"
      - "断点前后系数对照表"
      - "R² 时序（看模型解释力稳定性）"

  step3_risk_parity:
    description: |
      基于 FM 保留的因子做 Risk Parity 合成。
      等风险贡献（ERC）替代等权 z-score：
        目标：每因子对组合方差贡献相等
        约束：sum(w_i) = 1, w_i ≥ 0
      实现方法：
        (a) 每调仓日横截面上计算各因子 z-score
        (b) 用过去 60 个交易日各因子 top-quintile 多头组合收益序列估协方差 Σ
        (c) 求解 ERC 权重 w* = argmin sum_i sum_j (w_i*(Σw)_i - w_j*(Σw)_j)^2
        (d) 合成分 = sum_i w_i * zscore(F_i)
        (e) 选 top-50，按合成分 z-score 加权
    baseline_comparison: |
      Baseline: P1-F2F5-EP-LowVol-2026Q2-v1（0.5*z(F2) + 0.5*z(F5)）
      对比维度：总收益 / 年化 / Alpha / Sharpe / 最大回撤 / 换手率
    outputs:
      - "Risk Parity 时变权重序列（看权重稳定性）"
      - "RP 组合 vs Baseline 净值曲线对照"
      - "RP 组合 vs Baseline 收益/风险指标对照表"
```
