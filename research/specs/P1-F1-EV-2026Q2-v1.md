## 假设

```yaml
hypothesis: |
  EV<0（净现金覆盖）因子在 A 股具有显著的正向选股能力——市场对持有超额现金
  的低负债公司系统性低估，这些公司的股价在跨周期中回归基本面价值。
economic_intuition: |
  当 EV = 市值 + 总负债 − 现金及等价物 < 0 时，市场对公司的经营资产给出了
  负定价。此超额收益的来源有两层：
  
  (1) **清算价值溢价（Graham Net-Net 继承）**：Mohanty & Oxman (2026) 50年美股
      实证（1969-2019）表明，NCAV 策略在控制 Fama-French 五因子 + 流动性 + 一月
      效应后仍有 1.09%/月（13.9%/年）的显著 alpha，且非小公司效应。
  
  (2) **行为偏差（neglect/pessimism）**：市场对"账面现金多但业务平淡"的公司
      存在系统性忽视（neglect），对其经营资产过度悲观，给出低于清算价值的定价。
  
  **回应 thinking-prompt.md 的深层追问**：
  
  a) **壳价值污染**：2019 注册制前，大量小市值 EV<0 股票的"价值回归"来源于
     借壳上市预期而非经营层面价值回归。2016 借壳新规后市值最小 30% 股票年化
     收益从 28%→-5%（CSDN 2026 实证）。本 spec 设 Gate 1 规则 6（壳价值净化：
     报告剔除小市值后 IC）和规则 7（以 2019.06 为断点分段报告 IC）应对。
  
  b) **受限资金辨析**：聚宽 `cash_equivalents` 字段包含受限资金（保证金、专项
     账户、募集资金专户），可能高估可用现金。本 spec 采用 theory-framework §5.1
     的口径：净现金 = 货币资金 + 交易性金融资产 + 附注定期存款 − 有息负债；
     同时保留 `cash_equivalents` 单一口径作为对照组，偏差记录标注差异方向。
  
  c) **A+H 跨市场问题**：AH 溢价差异使"净现金 > 市值"在 A 股更罕见。本 spec
     仅覆盖 A 股，不涉及 H 股。若 A+H 同时满足条件，按 A 股交易；不额外论证
     溢价合理性（待 Phase 2 相关性分析时作为副发现上报）。
  
  d) **现金质量**：是否需挖财报附注定期存款。聚宽字段 `cash_equivalents` 漏算
     附注中"一年内到期的非流动资产"等类现金项目。本 spec 在定义的"数据缺失"
     字段中标注此限制，偏差记录中上报"可能低估净现金"的方向与影响。
  
  **回应 theory-framework §5.1 的精细化建议**：
  
  - 净现金计算扩展为：货币资金 + 交易性金融资产 + 附注定期存款 − 有息负债
  - 新增"流动比率 > 1.5"过滤条件（流动资产 ≈ 流动负债 时现金不能全扣）
  
  **回应 theory-framework §5.2 的必跑变体**：
  
  - 壳价值剔除版：注册制前 20 亿 / 注册制后 30-50 亿市值过滤
  - 本 spec 设 V1（基线版）与 V2（壳价值剔除版）两组变体
failure_condition: |
  明确失败条件：
  - IC_IR < 0.2（连续两轮则放弃该因子方向）
  - Q1 多头组年化收益未能跑赢基准
  - Q1-Q5 分组不单调
  - 三样本方向不一致
  - 壳价值剔除后 IC 归零或方向反转
  - 2019.06 前后分段 IC 方向相反
```

## 因子定义

```yaml
factors:
  - name: factor_ev_negative
    type: binary
    formula: >
      基线版（V1）：
        cash_available = balance.cash_equivalents
        interest_bearing_debt = balance.total_liability - 
          (balance.accounts_payable + balance.advances_from_customers +
           balance.wages_payable + balance.taxes_payable +
           balance.other_current_liabilities)    # 非有息负债剔除
        ev = valuation.market_cap + interest_bearing_debt - cash_available
        factor_ev_negative = 1 if ev < 0 else 0
    
      精细化版（V1 内含校验）：
        cash_available_ext = balance.monetary_funds + 
          balance.financial_assets_held_for_trading + 
          balance.time_deposits_note                     # 附注定期存款（若不可得则只取前两项）
        ev_ext = valuation.market_cap + interest_bearing_debt - cash_available_ext
    
      过滤条件（V1）：
        current_ratio = balance.total_current_assets / balance.total_current_liabilities
        仅保留 current_ratio > 1.5 的股票
    fields:
      market_cap: valuation.market_cap
      total_liability: balance.total_liability
      cash_equivalents: balance.cash_equivalents
      monetary_funds: balance.monetary_funds
      financial_assets_held_for_trading: balance.financial_assets_held_for_trading
      total_current_assets: balance.total_current_assets
      total_current_liabilities: balance.total_current_liabilities
      accounts_payable: balance.accounts_payable
    notes: |
      - 现金质量待校验：聚宽 cash_equivalents 含受限资金，方向为高估信号强度
      - 有息负债估算：总负债 − 无息流动负债（应付/预收/应付职工薪酬/应交税费/其他流动负债）
      - 流动比率 > 1.5 过滤：防止"流动资产≈流动负债"时扣掉经营所需现金
      - 附注定期存款：若聚宽字段不可得，在偏差记录中标注"未挖附注，可能低估净现金"

variants:
  - id: V2
    name: 壳价值剔除版
    hypothesis: 剔除小市值后，EV<0 的纯基本面收益仍显著为正
    filter: |
      - 2014-01 至 2019-05 期间：剔除市值 < 20 亿的股票
      - 2019-06 至 2026-06 期间：剔除市值 < 30 亿的股票
    notes: 注册制前后壳价值阈值不同（theory-framework §5.2）
```
