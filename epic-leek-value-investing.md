# “史诗级韭菜”深度价值投资体系：从理论到量化实现


## 第一章：理论基础与核心理念

### 1.1 投资哲学总纲

B站财经UP主“史诗级韭菜”的投资风格被评价为纯正的 **“格系捡烟蒂”流派**。其核心投资哲学可概括为一句话：**长期的超额收益，来源于利用市场的恐惧与厌恶情绪进行逆向投资。**

作者认为，人很容易因主观偏见（如肤色、地域、宏观数据）而看衰一个行业，但那些真正优质的公司本身并未衰退，最终股价会回归其内在价值。他深谙市场的极端情绪规律——当市场全面否认一个行业时，恰恰是深度价值投资者应当关注的时刻。

### 1.2 市场常见的四种错误认知

作者在2026年6月25日的动态中，尖锐地指出了当前市场上他认为极不理性的四种典型观点：

1. **忽视净现金价值**：认为账面上的巨额现金和零负债毫无价值，视而不见。
2. **过度要求股息率**：认为5-6%的股息率太低，必须达到8-10%才值得买入。
3. **极端分化定价**：认为科技股“不言顶”，可以无限上涨；而消费股即使估值极低，也依然被认为高估。
4. **认为“这次不一样”**：认为AI将重塑一切，所以当前的高估值水平是完全合理的，历史规律不再适用。

### 1.3 五大核心投资原则

作者通过长期实践，总结出五条铁律：

| 原则 | 核心内容 | 底层逻辑 |
|------|---------|----------|
| **原则一：财务稳健** | 投资于长期现金不断积累、且债务从未增加的企业 | 现金是抵御寒冬的棉衣，低负债是生存的底线 |
| **原则二：重视股东回报** | 投资于5-10年内不向市场融资，且有回购或大股东增持的企业 | 不稀释股东权益，真金白银回馈市场 |
| **原则三：持续分红** | 公司有长达5-10年以上的稳定分红记录 | 长期分红是利润真实性的最好证明 |
| **原则四：逆境抗压** | 在最困难的环境下依然能保持增长 | 压力测试下才能看出真正的护城河宽度 |
| **原则五：护城河价值** | 某些公司的现金流具备“类债”属性，其信用本身就很值钱 | 稳定的现金流等价于高等级债券 |

### 1.4 最终选股“四项铁律”

在上述原则基础上，作者进一步提炼出简单粗暴的筛选标准：

- **EV为负**：企业持有的净现金（现金减负债）大于其总市值。这意味着市场给公司经营资产贴上了“负价格”标签。
- **盈利能力强**：每年稳定赚取的利润超过市值的10%（即市盈率PE < 10）。
- **持续回馈股东**：持续、大额地进行分红或股份回购。
- **背景雄厚**：有国家控股作为背书，确保极端情况下的生存能力。

作者最后点出**商业的本质**：是赚取自由的现金，而不是拥有一堆厂房和设备。


## 第二章：实战历史案例——逆向投资的验证

### 2.1 辉煌战绩：在极端恐慌中精准买入

作者在多个行业遭遇系统性恐慌时逆势布局，这些操作在当时均遭到市场多数人的强烈反对和嘲笑，但最终都证明了他的判断：

**案例一：金融反垄断时期——买入腾讯控股**

在互联网平台经济遭遇史无前例的监管风暴、市场普遍担心“巨头时代终结”之际，作者选择逆势重仓买入腾讯。彼时市场情绪极度悲观，腾讯估值一度大幅压缩，政策不确定性笼罩全市场。事后看，腾讯的经营基本面并未因监管而发生根本性恶化，核心社交生态和游戏基本盘依然稳固，股价随后迎来大幅修复。此役成为作者最具代表性的成功案例之一。

**案例二：房地产崩溃衰退时——买入香港地产公司**

在内地房地产行业债务危机蔓延、恒大等巨头暴雷、市场对地产股唯恐避之不及时，作者并未被恐慌裹挟，而是冷静分析后选择买入以长实集团为代表的香港地产公司。作者曾公开指出，长实集团净负债率仅4-5%，财务极其稳健，但市净率（PB）却依然只有0.3倍左右。市场因宏观悲观情绪给出了严重过度折价，而这些公司本身的财务质量远好于市场预期。随着时间推移，市场重新认识到了这些公司的价值。

**案例三：消费持续衰退时——买入蒙牛、海底捞**

在消费数据持续低迷、市场普遍看衰消费板块、认为消费降级将摧毁品牌企业时，作者逆势买入蒙牛乳业和海底捞。从后续表现看，蒙牛股价从约9倍市盈率区域涨至15倍，海底捞股价涨幅更为显著。这些公司的品牌心智占领能力和经营韧性在市场情绪回暖后得到了资本市场的重新定价，验证了消费龙头在逆境中的生存能力。

**案例四：美国生物制裁法案时——买入药明生物**

在生物医药板块遭遇美国《生物安全法案》冲击、市场担心行业前景被彻底摧毁时，作者逆势买入药明生物。尽管买入后过程中波动剧烈，股价一度大幅下探，但药明生物凭借其CRDMO（合同研究、开发及生产组织）模式的全球稀缺性、难以替代的工程师红利，以及充沛的未完成订单储备（2025年底高达237亿美元），最终股价实现了盈利修复，证明了真正具备全球竞争力的企业不会因政治风波而丧失价值。

### 2.2 坦承失利：策略的局限性

作者在B站动态中极其坦诚地承认：“这段时间我和指数差不多是同步亏损的，没有战胜市场。” 这表明其策略并非万能钥匙：

- **系统性风险面前难以独善其身**：即使选股逻辑完全正确，在市场整体估值大幅压缩的流动性危机中，组合仍会跟随指数同步下跌，需要极强的心理承受能力。
- **逆向布局的时机陷阱**：买入过早会导致短期巨幅浮亏。以药明生物为例，即便最终结局是盈利，过程中也要长时间承受制裁法案带来的剧烈波动和心理煎熬。
- **实盘公开的巨大压力**：作者因在B站展示完整持仓讲解而被平台下架视频，最终只能通过付费私密动态分享实盘。这侧面反映了长期坚持逆向投资在公开流量场景下面临的合规压力和非议压力。

### 2.3 案例的经验总结

| 核心启示 | 具体说明 |
|---------|----------|
| **耐心是最大的成本** | 逆向布局后可能需要等待数月甚至数年才能迎来价值回归，其间股价可能继续下跌 |
| **财务质量是不可逾越的底线** | 只有真正低负债、现金充沛的公司才能熬过漫长的行业寒冬 |
| **警惕“这次不一样”的幻觉** | 每次恐慌都有看似无懈可击的理由，但商业的本质规律从未改变 |
| **分散是对抗不确定性的武器** | 即便逻辑再正确，单一标的仍可能面临不可预知的黑天鹅事件，适度分散是必要的 |


## 第三章：量化因子体系——从理念到数学公式

将上述五大原则和“四项铁律”转化为可计算的量化因子，是实现系统化投资的必经之路。

### 3.1 因子一：净现金覆盖因子（对应“EV为负”）

**理论基础**：当企业持有的净现金大于总市值时，意味着市场对这家公司的经营资产给出了负定价——相当于“免费”甚至“倒贴”获得其主营业务。这是“烟蒂股”投资的最极端形态。

**量化定义**：

$$EV = \text{总市值} + \text{总负债} - \text{现金及现金等价物}$$

**筛选条件**：$EV < 0$，即 $\text{现金及现金等价物} > \text{总市值} + \text{总负债}$

**聚宽数据字段**：
- 总市值：`valuation.market_cap`
- 总负债：`balance.total_liability`
- 现金及现金等价物：`balance.cash_equivalents`

**注意事项**：跌破净现金只代表公司清算价值的底线。若公司持有的是难以估值的无形资产（如品牌、专利）或重资产（如矿产），单纯看净现金可能会造成价值低估误判。

### 3.2 因子二：盈利收益率因子（对应“年利润 > 市值10%”）

**理论基础**：盈利收益率是市盈率（PE）的倒数。要求盈利收益率 > 10%即PE < 10，属于经典的低估值价值因子。巴菲特曾多次强调，盈利收益率是衡量股票是否便宜的核心指标之一。

**量化定义**：

$$\text{盈利收益率} = \frac{\text{净利润}}{\text{总市值}}$$

**筛选条件**：$\text{盈利收益率} > 10\%$，即 $\text{PE} < 10$

**聚宽数据字段**：
- 净利润：`income.net_profit`
- 总市值：`valuation.market_cap`
- 市盈率：`valuation.pe_ttm`

**有效性验证**：历史回测数据显示，低市盈率分位数单因子组合表现优异，相对沪深300的超额收益可达409.30%，年化收益率17.34%。

### 3.3 因子三：股东回报综合因子（对应“持续分红+回购”）

**理论基础**：仅使用股息率因子选股容易导致严重的行业偏离（如集中于银行、煤炭等传统行业）。更全面的做法是综合考虑分红、回购和债务偿还三项指标，构建综合股东回报率。

**量化定义**：

$$\text{股息率} = \frac{\text{净利润} \times \text{分红率}}{\text{总市值}}$$

$$\text{回购率} = \frac{\text{回购金额}}{\text{总市值}}$$

$$\text{综合股东回报率} = \text{股息率} + \text{回购率}$$

**筛选条件**：
- 过去三年连续现金分红，股利支付率在0-1之间
- 过去一年有回购公告或回购实施记录

**聚宽数据字段**：
- 股息率：`valuation.div_yield`
- 分红数据：`balance.dividend_payable`
- 回购数据：需通过聚宽事件数据接口获取

**有效性验证**：
- 股息率TTM因子在沪深300样本中IC均值为**0.0374**，IC_IR为0.2749，IC胜率64.71%；在中证500样本中IC均值为**0.0494**，IC_IR为0.4484，IC胜率64.96%。
- “股息率50”策略（按过去三年平均股息率排序取前50只）在2014-2025年的**年化收益率为16.00%**，同期沪深300全收益仅8.56%。
- 在净回购或债务净偿还的股票池中筛选高股息股票，组合**年化收益20.0%**，夏普比率0.95，平均每年相比中证红利全收益指数获得约**9%的超额收益**。

### 3.4 因子四：国企背景因子（对应“国家控股”）

**理论基础**：国家控股的企业在融资便利性、产业政策支持、极端风险兜底等方面具有天然的制度性优势，是“逆境抗压”原则的重要保障。

**量化定义**：

$$\text{国企虚拟变量} = \begin{cases} 1, & \text{实际控制人为国务院国资委/地方政府} \\ 0, & \text{其他} \end{cases}$$

**筛选条件**：`实际控制人类型 in ["国务院国有资产监督管理委员会", "地方国有资产监督管理委员会", "地方政府"]`

**聚宽数据字段**：
- 实际控制人：`valuation.actual_controller`

### 3.5 因子五：财务质量因子（对应“财务稳健+逆境抗压”）

**理论基础**：低负债、现金流充沛的公司具备更强的抗风险能力，能够在经济下行周期中存活并逆势扩张。

**量化定义**：

$$\text{资产负债率} = \frac{\text{总负债}}{\text{总资产}}$$

$$\text{每股经营现金流} = \frac{\text{经营活动现金流净额}}{\text{总股本}}$$

**筛选条件**：
- 资产负债率低于行业平均水平（或市场中位数）
- 每股经营现金流为正值且持续增长

**聚宽数据字段**：
- 资产负债率：`balance.debt_to_assets`
- 经营活动现金流净额：`cash_flow.operating_cash_flow`
- 总股本：`valuation.total_shares`

### 4.2 因子计算函数

```python
def calculate_all_factors(df):
    """
    计算五大因子值，并生成基础评分
    """
    # -------- 因子一：EV为负 --------
    df['ev'] = df['market_cap'] + df['total_liability'] - df['cash_equivalents']
    df['factor_ev_negative'] = (df['ev'] < 0).astype(int)
    
    # -------- 因子二：盈利收益率 > 10%（即PE<10） --------
    df['earnings_yield'] = df['net_profit'] / df['market_cap']
    df['factor_high_ey'] = (df['earnings_yield'] > 0.10).astype(int)
    # 作为替代，可直接使用PE条件
    df['factor_low_pe'] = (df['pe_ttm'] < 10).astype(int)
    
    # -------- 因子三：高股息（取全市场前30%分位） --------
    # 注意需剔除股息率为0或负的异常值
    valid_div = df[df['div_yield'] > 0]['div_yield']
    if len(valid_div) > 0:
        high_div_threshold = valid_div.quantile(0.70)
    else:
        high_div_threshold = 0
    df['factor_high_div'] = (df['div_yield'] >= high_div_threshold).astype(int)
    
    # -------- 因子四：国企背景 --------
    state_owners = ['国务院国有资产监督管理委员会', '地方国有资产监督管理委员会', '地方政府']
    df['factor_state_owned'] = df['actual_controller'].isin(state_owners).astype(int)
    
    # -------- 因子五：财务质量 --------
    # 低杠杆：资产负债率低于市场中位数
    median_debt = df['total_liability'] / df['total_assets'].median()
    df['debt_to_assets'] = df['total_liability'] / df['total_assets']
    df['factor_low_leverage'] = (df['debt_to_assets'] < median_debt).astype(int)
    
    # 正经营现金流
    df['factor_positive_cf'] = (df['operating_cash_flow'] > 0).astype(int)
    
    return df
```

### 4.3 Z-score标准化与综合评分

```python
def composite_score(df):
    """
    对各项因子进行Z-score标准化后等权合成综合得分
    """
    # 需要评分的因子列
    factor_columns = [
        'factor_ev_negative', 
        'factor_low_pe',          # 或 factor_high_ey
        'factor_high_div', 
        'factor_state_owned', 
        'factor_low_leverage', 
        'factor_positive_cf'
    ]
    
    # 进行Z-score标准化 (注意避免标准差为0)
    for col in factor_columns:
        mean = df[col].mean()
        std = df[col].std()
        if std > 0:
            df[col + '_zscore'] = (df[col] - mean) / std
        else:
            df[col + '_zscore'] = 0  # 所有值相同的情况
    
    # 等权合成综合得分
    zscore_cols = [col + '_zscore' for col in factor_columns]
    df['composite_score'] = df[zscore_cols].mean(axis=1)
    
    return df
```

    if df.empty:
        return
    
    # 剔除关键数据缺失的股票
    df = df.dropna(subset=['market_cap', 'pe_ttm', 'net_profit', 'total_liability'])
    
    # 3. 计算因子和综合得分
    df = calculate_all_factors(df)
    df = composite_score(df)
    
    # 4. 按综合得分降序排列，取前N只
    df = df.sort_values('composite_score', ascending=False)
    selected_codes = df.head(g.stock_num)['code'].tolist()
    
    log.info('本期选中股票数量: %d' % len(selected_codes))
    log.info('综合得分前5名: %s' % selected_codes[:5])
    
    # 5. 执行调仓
    # 先卖出不在新持仓中的股票
    for stock in context.portfolio.positions:
        if stock not in selected_codes:
            order_target_percent(stock, 0)
            log.info('卖出 %s' % stock)
    
    # 再买入新持仓（等权配置）
    target_weight = 1.0 / len(selected_codes) if len(selected_codes) > 0 else 0
    for stock in selected_codes:
        order_target_percent(stock, target_weight)
        log.info('买入 %s, 权重 %.2f%%' % (stock, target_weight * 100))

def is_new_stock(code, date, days):
    """判断是否为上市不满days天的次新股"""
    info = get_security_info(code)
    if info is None:
        return True
    start_date = info.start_date
    current_date = datetime.datetime.strptime(date, '%Y-%m-%d')
    if (current_date - start_date).days < days:
        return True
    return False

## 第五章：注意事项与风控体系

### 5.1 关键风险提示

1. **因子衰减风险**：任何因子的有效性都可能随时间衰减，尤其在市场风格剧烈切换时可能短期失效。

2. **A股市场特殊性**：A股股东回报结构（分红为主、回购规模小）与美股差异较大，不能简单照搬海外经验。

3. **财务数据时滞**：财务数据存在披露延迟（季报滞后约1个月，年报滞后约4个月），回测时必须使用当时可得数据，严格避免未来函数。

4. **极端值处理**：需对财务数据进行异常值处理（如剔除净利润为负、市值过小、资产负债率超过100%等）。

5. **行业集中度风险**：单一因子（如股息率）可能导致严重的行业偏离（如银行、煤炭），需结合行业中性化处理或设置行业上限。

6. **流动性风险**：小市值股票的流动性不足可能在实际交易中产生巨大滑点，回测时应剔除日均成交额过低的股票。

### 5.2 建议的增强措施

| 措施 | 具体做法 |
|------|---------|
| **行业中性化** | 在每个行业内部分别计算因子得分，避免因子暴露于特定行业 |
| **市值中性化** | 对因子值进行市值回归取残差，剔除市值因子的干扰 |
| **动态权重调整** | 根据各因子的IC（信息系数）衰减情况动态调整权重 |
| **止损机制** | 设定单只个股-20%或-30%的硬性止损线 |
| **宏观择时（可选）** | 在极端高估的市场整体估值水平下降低总仓位 |
